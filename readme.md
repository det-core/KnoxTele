## Quick Deploy

### Prerequisites
- Node.js 18+
- NPM/Yarn
- PM2 (for production)
- Telegram Account (for pairing)

### Deployment Steps

```bash
# 1. Clone repository
git clone https://github.com/det-core/KnoxTele.git
cd KnoxTele

# 2. Install dependencies
npm install

# 3. Configure environment
cp .env.example .env
# Edit .env with your settings

# 4. Start the bot
npm start

# Or use PM2 for production
pm2 start index.js --name knox-md
pm2 save
pm2 startup
```
